Atsakymas pateikiamas per console'ę

Aprašymas:

  • Consolėje sukuriame naują Calculator Object'ą. Pvz.: const c = new Calculator()
  • Naršyklė paprašys įvesti du skaičius a ir b
  • Kad gauti sumą rašome, pvz.: c.sum()
  • Kad gauti skirtumą rašome, pvz.: c.subtraction()
  • Kad gauti daugybą rašome, pvz.: c.multiplication()
  • Kad gauti dalybą rašome, pvz.: c.division()
